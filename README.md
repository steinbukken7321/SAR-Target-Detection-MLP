# SAR-Target-Detection-MLP
Desenvolvimento de um sistema de detecção de alvos terrestres em imagens de radar de abertura sintética (SAR) utilizando o dataset CARABAS-II. O projeto foca na implementação de Redes Neurais (MLP) otimizadas por técnicas avançadas de Processamento Digital de Imagens para filtragem de ruído e realce de alvos em cenários de vegetação densa
